# Weather App

تطبيق يعرض حالة الطقس في أي مدينة باستخدام واجهة برمجة التطبيقات (API). يمكنك البحث عن المدينة والحصول على تفاصيل مثل درجة الحرارة، الطقس الحالي، والرطوبة.

## التقنيات المستخدمة

- HTML
- CSS
- JavaScript
- Weather API

## طريقة الاستخدام

1. افتح `index.html`.
2. اكتب اسم المدينة للحصول على معلومات الطقس.
